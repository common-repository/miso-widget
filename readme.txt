== Description ==

With this Miso widget you can add your gomiso.com checkin list to your wordpress blog.

== Installation ==

1. Download the zip file and extract the contents.
2. Make sure you're running WordPress version 3.0 or better. It won't work with older versions.
3. Upload the 'miso-widget' folder to your plugins directory (plugins/miso-widget).
4. Activate the plugin through the 'plugins' page in WP.
5. See widget options to set your login name and the cover visibility.

== Frequently Asked Questions ==

= Can I use my login name or I have to find out whats my user ID is? =

* In the User ID/Login name field you can easily set your gomiso.com login name.
